# Python OOP Examples

A small collection of Python example scripts that demonstrate core Object-Oriented Programming (OOP) concepts.

## Overview

This repository contains short, focused examples for: classes, constructors, inheritance (single, multiple, and multi-level), encapsulation, polymorphism, abstraction, and methods. Each file is a standalone script you can run to see the concept in action.

## Prerequisites

- Python 3.8 or newer

## Run an example

From the repository root, run a file using Python. For example:

```bash
python polymorphism.py
```

Replace `polymorphism.py` with any filename from the list below.

## Files

- `abstract.py`: Example showing abstraction using abstract base classes.
- `bank.py`: Small illustrative example (bank/account) using OOP.
- `class.py`: Basic class definition and usage.
- `constructor.py`: Demonstrates `__init__` constructors and initialization.
- `encapsulation.py`: Shows encapsulation and access control (name mangling/private attributes).
- `inheritance.py`: Single inheritance example.
- `multi_level_inheritance.py`: Multi-level inheritance example.
- `multiple_inheritance.py`: Multiple inheritance example.
- `method.py`: Examples of instance, class, and static methods.
- `polymorphism.py`: Demonstrates polymorphism via method overriding and duck typing.

## Notes

- Each script is simple and intended for learning; modify them to experiment.
- No external dependencies are required.
